# C Programming Repository

This repository is created for practicing and organizing C programming codes.  
It contains programs based on basic concepts, loops, conditional statements, arrays, functions, pointers, strings and many more.  
Each program is saved topic-wise to make revision easy and fast.

## Purpose
- To store all C programs in an organized structure
- To improve learning and coding skills
- To prepare for practical exams, viva and interviews

## Folder Structure
/Basic-Programs  
/Loops  
/Conditionals  
/Arrays  
/Functions  
/Pointers  
/Strings  
/Notes  
/Output-Screenshots

## How to Use
- Open any folder and run programs in the C compiler (Turbo C / GCC).
- Check Output screenshots for reference.
- Add new programs daily to improve consistency.
